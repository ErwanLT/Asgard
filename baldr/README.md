## Description
Baldr est un module de test d'intégration conçu pour simuler des conditions réelles d'utilisation des autres modules du système. Il fournit un environnement de test complet avec une application de démonstration qui permet de valider le bon fonctionnement des différents composants dans un contexte proche de la production.

## Objectifs
- Fournir un environnement de test réaliste
- Simuler les interactions entre les différents modules
- Permettre la validation des fonctionnalités dans un contexte d'utilisation réelle
- Faciliter la détection des problèmes d'intégration

## Fonctionnalités
- Application de test complète
- Scénarios de test prédéfinis
- Simulation de données réalistes
- Outils de monitoring et de débogage

## Utilisation
1. Déployer l'application de test
2. Configurer les paramètres de test selon vos besoins
3. Exécuter les scénarios de test
4. Analyser les résultats

## Configuration
Les paramètres de configuration se trouvent dans le fichier `application.properties` ou `application.yml`.

## Dépendances
- Java 17 ou supérieur
- Spring Boot
- Autres dépendances listées dans le `pom.xml`
